# GenAI Conversational Analytics Dashboard

## Problem
Business users struggle to extract insights from MongoDB data and dashboards
because they require technical knowledge of queries and charts.

## Solution
A GenAI-powered analytics assistant that allows users to ask business
questions in natural language and automatically generates:
- MongoDB queries
- Visual dashboards
- Business-friendly insights

## Example
User: "Show sales trends for last quarter"
System:
- Generates MongoDB aggregation
- Displays a line chart
- Explains trends in plain English
